### How to Install / Use
1. Download the latest release package from the link below.
2. Extract the toolkit to a secure local directory (e.g., `C:\Tools\WTAPT`).
3. Run the `init_performance.exe` as Administrator to calibrate system offsets.
4. Select your desired tactical overlay preset from the configuration menu.
5. Launch Warzone and observe the performance telemetry overlay.
6. Adjust visual clarity filters using the hotkey `Ctrl + Shift + T`.
